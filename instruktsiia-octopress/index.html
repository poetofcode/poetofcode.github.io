Памятка Octopress

Четверг, 22 Мая 2014

Постоянно забываю как настраивать и как пользоваться этим блогом, поэтому решил оставить тут памятку для самого себя

Установка окружения

  • Ставим RVM:

    curl -L https://get.rvm.io | bash -s stable –ruby

  • Ставим руби:

    rvm install 1.9.3

    rvm use 1.9.3

    rvm rubygems latest

Важно: для запуска из терминала rvm и всего остального нужно в настройках терминала поставить галку “Run command as a login shell”

Настройка деплоймента

  • git clone https://github.com/poetofcode/poetofcode.github.io.git
  • cd poetofcode.github.io.git
  • rake setup_github_pages

Команды rake

  • Новый пост: rake new_post[“имя-поста”]
  • Сгенерировать блог: rake generate
  • Деплой: rake deploy
  • И то и другое: rake gen_deploy

Синтаксис Markdown

Заголовки:
# Заголовок 1
## Заголовок 2
### Заголовок 3

Цитата:
> Текст цитаты

Блок кода
Просто блок: ставим таб или 4 пробела перед тектом
С подсветкой синтаксиса для кода (в примере апострофы разделены пробелом, но пробелов быть не должно)
` ` ` html
Ваш код
` ` `

Выделение текста:
** жирный **
* курсив *

Ссылки:
Обычная сслыка: [This link](http://example.net/)
На изображение: ![Alt text](/path/to/img.jpg)

Подробнее тут: http://daringfireball.net/projects/markdown/

Flickr

{ % flickr_image id [preview-size [alignment [caption]] % }
{ % flickr_set id [preview-size [desc|nodesc]] % }

Примечание: без пробелов между { и %
preview-size
o : “Original”, no maximum dimension
b : “Large”, 1024px
z : “Medium 640”, 640px
n : “Small 320”, 320px
m : “Small”, 240px,
t : “Thumbnail”, 100px
q : “Large Square”, 150px
s : “Square”, 75px
alignment
The alignment is specified as left, right, or center, like the rest of Octopress.

Подробнее: https://github.com/neilk/octopress-flickr#how-to-use-the-tags
